# Stellar Template

Stellar is a premium, generic static website template designed with modern web standards, beautiful dark mode UI, and glassmorphism elements. It is optimized for seamless deployment to GitHub Pages using GitHub Actions.

## Features

- **Premium Design:** Curated dark mode color palette, modern typography (Outfit font), and glassmorphism cards.
- **Micro-Animations:** Smooth hover effects and intersection-observer based scroll animations for an engaging experience.
- **Lightning Fast:** Built purely with HTML, CSS, and vanilla JS. No heavy dependencies or frameworks.
- **Automated Deployment:** Pre-configured GitHub Actions workflow to easily publish the site to GitHub Pages.
